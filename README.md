# 2010-2020ChinaTyphoon
A data visualization project about the typhoons that affected China from 2010 to 2020
运行方法
1. 打开终端，定位到此文件夹
2. 输入： streamlit run 2010-2020中国台风可视化系统.py
3. 将运行得到的网址打开即可
